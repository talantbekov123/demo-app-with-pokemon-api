## Installation


## Running the app

```bash
# development
$ npm run start

$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash

$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
